# Release Notes - Lap v0.1.10

Lap v0.1.10 is a feature and usability update that improves browsing, organization, and day-to-day workflow performance.

## 🚀 Highlights

### 🛡️ Stability and Scanning Improvements
- **Crash fix included:** This release includes the recent crash fix and related recovery improvements.
- **Background scanning:** Indexing now runs silently in the background, with a smoother scan flow and improved library switching behavior.

Special thanks to [Fischmuetze](https://github.com/Fischmuetze), [themacguru](https://github.com/themacguru), and [JochenHiller](https://github.com/JochenHiller) for reporting, testing, and verifying the crash issue tracked in [Issue #13](https://github.com/julyx10/lap/issues/13).

### 🔠 Custom Font Size Support
- **Adjustable interface font size:** Lap now supports multiple built-in font size levels, making the app more comfortable to use on different displays.

### 📷 Lens Grouping
- **Group photos by lens:** The Camera panel now supports lens-based grouping in addition to camera grouping.
- **Re-scan may be needed:** Some existing libraries may need a re-scan for lens data to appear correctly because lens EXIF metadata is often inconsistent or missing.

### ⌨️ Shortcuts and Selection Workflow
- **Improved shortcuts:** Common actions like favorite, rating, tagging, commenting, rotating, and showing file info are now easier to access.
- **Better multi-select workflow:** Selection mode and batch actions were refined for faster organization work.

### 🏷️ Smart Tags and Metadata
- **Smart tags expanded:** Smart tags now include 8 practical categories for everyday photo collections and photography workflows.
- **Unrated filter:** Rating views now support an unrated state for photos without a rating.

### 🎨 UI Improvements
- **Font, panel, and info view refinements:** Multiple panels, empty states, metadata displays, and toolbar interactions were polished.

Thank you for using Lap!
