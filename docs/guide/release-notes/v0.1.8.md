# Release Notes - Lap v0.1.8

Lap v0.1.8 expands editing and comparison workflows, makes selection and dedup cleanup more practical, and improves Windows and large-library reliability.

## 🚀 Highlights

### 🖼️ Editing, Adjustments & Compare
- **Dedicated adjust workflow:** Lap now separates quick image editing and manual adjustment workflows more clearly, with a focused adjust panel for brightness, contrast, saturation, hue, blur, presets, and histogram-guided tuning.
- **Save as new file:** Edited or adjusted images can now be saved as a new file, indexed automatically, and inserted back into the current file list without a manual refresh.
- **Image comparison mode:** The viewer can now open side-by-side comparison mode and keep navigation aligned for easier before/after or duplicate review.

### 🧭 Selection, File Info & Browsing
- **Dedicated selection panel:** Multi-selection actions now live in a dedicated right-side panel with clearer batch actions for move, copy, trash, favorite, rating, and tags.
- **Improved folder breadcrumbs:** Content titles, file info, and duplicate review now use consistent breadcrumb-style folder paths instead of full raw paths.
- **Better quick rename and file info editing:** File info now provides cleaner rename entry points, better inline editing behavior, and more focused folder/album presentation.

### 🧹 Dedup Cleanup & Library Workflow
- **Refined dedup panel:** Duplicate cleanup now has a more focused panel layout with compare-group actions, keep-file controls, and improved duplicate context.
- **More reliable duplicate deletion:** Duplicate removal flows were corrected so selected duplicates can be removed even when those files are not currently loaded in the visible content list.
- **Safer file list refresh behavior:** File list refreshes, loading states, and dedup rescans were adjusted to avoid unnecessary blank states and keep scans aligned with real list updates.

### 🪟 Platform & Viewer Fixes
- **Better Windows compatibility:** Windows file loading, MOV playback, frameless window behavior, and folder authorization were improved, including support for dynamically authorized library folders instead of broad static path access.
- **Smoother startup and library switching:** The app now avoids long blank-window states during startup, database load, and library switching by showing the shell earlier and refining loading feedback.
- **Viewer polish:** Quick preview, rotation handling, slideshow transitions, fullscreen status-bar behavior, and overlay badge placement were refined for more stable presentation.
